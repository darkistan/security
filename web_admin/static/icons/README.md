# PWA Іконки

Іконки для Progressive Web App «Система ведення змін охоронців».

## Необхідні файли

- `icon-192x192.png` — 192×192 px (Android)
- `icon-512x512.png` — 512×512 px (рекомендовано)
- `apple-touch-icon.png` — 180×180 px (iOS)

Додайте їх сюди вручну або скопіюйте з проєкту «Система заявок» (`web_admin/static/icons/`).

Без іконок PWA працює; запити на apple-touch-icon повертають 204.
